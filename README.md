# Blundr

[![GitLab Wiki Badge](https://img.shields.io/badge/GitLab-Wiki-d94a34.svg?logo=gitlab)](https://git.chalmers.se/courses/dit826/2025/team2/-/wikis)

**Blundr** is an AI-driven dating-coach system designed to help users improve their communication and interpersonal skills through simulated dating interactions. Users are matched based on preferences and engage in live video calls that the system performs emotion, speech, and transcription analysis on. After each session, a Large Language Model evaluates key conversational moments, rated with a [chess.com](https://www.chess.com) style scale and provides targeted feedback. 

*Developed as part of **DIT826 – Software Engineering for Data-Intensive AI Applications***
